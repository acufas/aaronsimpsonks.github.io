The Taxonomy of Numbers

There are a variety of numbers we encounter in mathematics and they can be classified into various sets. Here are a few of the most important ones.

  1. The Empty Set: ∅ = {} = {x | x != x}

    The set with no elements. The definition is a bit peculiar, but makes logical sense — the set of elements X such that X is not equal to itself is empty, since there is no number that is not equal to itself.

  2. The Natural Numbers: N = {1, 2, 3, ...}
  3. The Whole Numbers: W = {0, 1, 2, 3, ...}
  4. The Integers: Z = {..., -3, -2, -1, 0, 1, 2, 3, ...}
  5. The Rational Numbers: Q = {a/b | a ∈ Z and b ∈ Z}

    Rational numbers are the ratios of integers, assuming the bottom integer is nonzero. Furthermore, the decimal form of a rational number either ends (0.25) or repeats the same sequence (0.666666666666...).

  6. The Real Numbers: R = {x | x can be represented as a decimal}
  7. The Irrational Numbers: P = {x | x is a non-rational real number}

    Defined using the above set, an irrational number is a number that is non-rational and real. They are real because they can be represented as a decimal, and non-rational because their decimal representation does not end nor repeat. Examples are pi, the constant e, the square root of 2, and more.

  8. The Complex Numbers: C = {a + bi | a,b ∈ R and i = √−1}

    The set of all numbers that can be expressed in the form a + bi, where a and b are real numbers and i is defined as the square root of -1. Useful in physics and other forms of applied mathematics.

With some thought, this taxonomy gets a bit more interesting. You'll see that the set of Whole Numbers contains all the Natural Numbers. Furthermore, the Integers contain all the Whole Numbers, plus negative integers. Since every Integer can be represented by itself over 1, all Integers are Rational Numbers. Every Rational Number has a decimal form, so all Rational Numbers are Real Numbers. The Irrational Numbers are also Real Numbers, as stated in their definition. Finally, if we set b equal to 0, we can represent every one of these numbers in the form a + bi, so they all fall under the Complex Numbers.
